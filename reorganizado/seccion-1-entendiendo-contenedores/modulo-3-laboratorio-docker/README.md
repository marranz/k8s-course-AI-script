# Módulo 3: Laboratorio con Docker y Docker Compose

## Objetivo del Laboratorio

Este módulo es completamente práctico. Los estudiantes aplicarán todos los conceptos aprendidos en el Módulo 2 a través de ejercicios hands-on.

## Estructura del Laboratorio

1. [Ejercicios Básicos de Docker](./ejercicios-basicos.md)
2. [Ejercicios de Dockerfile](./ejercicios-dockerfile.md)
3. [Ejercicios de Volúmenes y Persistencia](./ejercicios-volumenes.md)
4. [Ejercicios de Redes](./ejercicios-redes.md)
5. [Ejercicios de Docker Compose](./ejercicios-docker-compose.md)
6. [Proyecto Final](./proyecto-final.md)

## Requisitos Previos

- Docker Desktop instalado o Docker Engine en un servidor Linux
- Editor de texto (VS Code recomendado)
- Conocimientos básicos de línea de comandos
- Acceso a internet para descargar imágenes

## Duración Estimada

- Ejercicios básicos: 30 minutos
- Ejercicios de Dockerfile: 45 minutos
- Ejercicios de volúmenes: 30 minutos
- Ejercicios de redes: 30 minutos
- Ejercicios de Docker Compose: 1 hora
- Proyecto final: 1-2 horas
