## Objetivos

- Comprender la necesidad de estándares en el ecosistema de contenedores
- Conocer la Open Container Initiative (OCI)
- Entender qué es containerd y su papel en el ecosistema
- Aprender la relación entre Docker, containerd y Kubernetes
- Practicar con containerd usando la herramienta `ctr`

## Conceptos Clave

- **OCI (Open Container Initiative):** Organización que define estándares abiertos para contenedores
- **OCI Image Format Specification:** Estándar para el formato de imágenes
- **OCI Runtime Specification:** Estándar para la ejecución de contenedores
- **Containerd:** Runtime de contenedores de alto nivel
- **runC:** Runtime de bajo nivel que ejecuta contenedores
- **CRI (Container Runtime Interface):** Interfaz estándar de Kubernetes para runtimes
